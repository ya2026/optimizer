# STEP 3D Web Tool

A pure front-end online 3D tool scaffold built with Nuxt 3, TypeScript, and Three.js.

## Tech Stack

- Nuxt 3
- Vue 3 Composition API
- TypeScript with strict mode
- Three.js

## Scripts

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run preview`

## Current Scope

This stage only includes:

- project initialization
- local Git repository setup
- strict TypeScript configuration
- three-column page layout
- placeholder UI regions for file import, 3D canvas, manual coloring, and export

Core STEP parsing, Three.js rendering, automatic preprocessing, manual face coloring, and export logic will be implemented in later steps.
